  1. What are the different data types in JavaScript?

    In JavaScript we have mainly two types of data type - 1. Primitive Data Types and 2. Non-primitive Data Type or Object
    Primitive Data Types - Primitive data types are passed by value. pass by value means, if we try to copy value of a Primitive data type variable in another variable, we can do like this 

                            let a = 3;
                        let b = a;
                        b= 5;
                        console.log(a) // output : 3
                        console.log(b) // output : 5
    Now if we try to change value of b as b=5, it will only change value of b not a.
    Reason behind pass by value is - both variable and its value is present in main memory.
