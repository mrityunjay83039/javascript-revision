Important Polyfills

  1. Polyfill of map function

    1. it takes an array
    2. it returns an array
    3. runs a callback function for each element of the array

    2.             
                Array.prototype.myMap = function(callback){
                    let res = [];
                    for(let i=0; i<(this.length); i++){
                        let op = callback(this[i]);
                        res.push(op);
                    }
                
                    return res;
                }

  2. Polyfill of filter function

    1. it takes an array 2. it returns an array 3. runs a callback function for each element of the array 

    3.                 
                    Array.prototype.myFilter = function(callback){
                        let res = [];
                        for(let i=0; i<(this.length); i++){
                            if(callback(this[i])){
                                res.push(this[i]);
                            }
                        }
                        return res;
                    }
